# J.A.R.V.I.S — AI Assistant

> Just A Rather Very Intelligent System

A sleek, Iron Man-inspired AI assistant built with voice recognition, text-to-speech, and free LLM models via Groq.

**Built and created by Aswathy Salikumar.**

---

## Features

- 🎙️ **Voice Input** — Speak to Jarvis using your microphone
- 🔊 **Text-to-Speech** — Jarvis speaks back to you
- 🧠 **Conversation Memory** — Remembers context within a session
- ⚡ **Multiple Free AI Models** — Powered by Groq (Llama, Mixtral, Gemma, DeepSeek)
- 🎨 **Iron Man HUD UI** — Animated reactor, scanline effects, futuristic design
- 💾 **API Key Stored Locally** — Your key never leaves your browser

## Getting Started

### 1. Get a Free Groq API Key
Go to [console.groq.com/keys](https://console.groq.com/keys) and generate a free key. No credit card needed.

### 2. Open the App
Visit the live demo or open `index.html` locally in Chrome.

### 3. Authenticate
Paste your Groq API key in the left panel and click **AUTHENTICATE**.

### 4. Talk to Jarvis
Type in the input box or click the **microphone** to speak.

## Local Setup

```bash
git clone https://github.com/aswathysalikumar28-lang/Jarvis.git
cd Jarvis
# Open index.html in Chrome
```

No install, no build step, no server needed.
